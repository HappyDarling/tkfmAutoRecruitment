## tkfmAutoRecruitment

This can make TenkafuMA Recruitment automatically (Only Korean)

## 설치 방법
1. 파이썬 3.7.0 설치 (반드시 PATH 체크박스 선택)  

2. Tesseract-OCR 설치 (C:\Program Files\Tesseract-OCR\tesseract)  

3. tkauto 파이썬 패키지 설치
    ```
    pip install tkauto
    ```
    
4.
